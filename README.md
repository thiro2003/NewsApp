# NewsApp

NewsApp is a React-based web application that fetches and displays the latest news articles from around the world. Users can view news by category and stay updated with top headlines.

## Features

- Display latest news articles
- Filter news by categories (e.g., technology, sports, business)
- Responsive design for desktop and mobile
- Clean and user-friendly interface
- Uses NewsAPI to fetch real-time news data

## Technologies Used

- **Frontend:** React.js, HTML, CSS, JavaScript
- **API:** [NewsAPI](https://newsapi.org/)
- **State Management:** React Hooks (`useState`, `useEffect`)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/thiro2003/NewsApp.git
```
2. Navigate into the project folder:
   cd NewsApp
3. Install dependencies:
   npm install
4. Add your NewsAPI key:
   Create a .env file in the root directory.
   Add your API key:
      npm start
Usage

Browse the latest news articles on the homepage.

Select a category to filter news.

Click on a news article to read the full story on the original site.
